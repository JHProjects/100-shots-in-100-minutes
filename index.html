About ►

100/100 Game
ver. 0.24 alpha
A simple drinking game, far supperior than your usual timer on your phone. The main functions are: Visual indications & group and personal statistics. This web application is made with stock JavaScript and CSS. It does not have any backend database (if you refresh, every data will be erased).

How-To-Play ►

Because there are still many bugs, you should follow theese instructions to avoid any softlocks and errors. 0. Never reload page unless you want to delete all progress. 1. Set up all your players before starting the timer. 2. Give all players every possible info (gender, shot-size, weight & AbV. are nescesary to count alcohol intoxication). 3. Then you can start the timer. 4. If you reset the timer, the player data will be broken, however you can pause and play the timer no problem.

Rules ►

If you don't get the rules you REALLY shouldn't play this game... You are either very underage or just rertarded (and in any case alcohol isn't very good for you).
—
Anyhow... It's really simple: Every 60 seconds you pour yourself a shot of (any) beer (20–60 ml) and gulp it down just as 59 changes to 0. The official challange is to get to 100 minutes (1h and 40min). If you want to destroy yourself, you can push even further.
